# merge_rules.md

## 통합 기준
- 같은 사건: 동일한 발표, 정책, 실적, 제품 출시, 규제 변화, 시장 이벤트를 다루는 경우
- 같은 논점: 같은 쟁점이나 해석 축을 반복하는 경우
- 같은 흐름: 하루 동안 반복적으로 나타난 방향성이 같은 경우
- 같은 해설 축: YouTube에서 길게 설명하는 논점이 X 반응이나 뉴스 핵심 내용과 직접 연결되는 경우

## 통합 우선순위
1. X와 뉴스가 모두 같은 흐름을 가리키는가
2. 뉴스 원문이 X 반응을 뒷받침하는가
3. X에서 반복되는 관점이 뉴스 핵심 내용과 연결되는가
4. YouTube 설명이 위 두 소스의 맥락을 보강하는가
5. 오늘 다시 볼 가치가 있는가

## 분리 기준
- 표면 키워드만 비슷하고 실제 사건이 다르면 분리한다.
- 같은 카테고리라도 지역, 산업, 정책 단계가 다르면 별도 cluster 또는 하위 메모로 다룬다.
- YouTube가 비슷한 키워드를 쓰더라도 설명 대상 사건이 다르면 억지로 합치지 않는다.
