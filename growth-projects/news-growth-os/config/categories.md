# categories.md

이 프로젝트는 공통 카테고리 기준으로 `../shared/config/categories.md`를 사용한다.
아래는 뉴스 기사 분류에서 자주 보는 세부 힌트다.

- 경제: 금리, 물가, 고용, 환율, 제조업, 에너지, 정책 변화
- 투자: 기업 실적, 가이던스, 시장 반응, 섹터 흐름, 위험자산, 자금 이동
- AI: 모델 출시, 제품 발표, API 정책, 기업 도입, 에이전트, 자동화
- 커리어: 채용, 조직 변화, 생산성 도구, 업무 방식, 개발 문화
- 기타: 위 카테고리에 바로 넣기 어렵지만 의미 있는 기사
