# AGENTS.md

## 프로젝트 목적
이 프로젝트는 X 기반 정보 수집과 저녁 분석을 통해
경제, 투자, AI, 커리어 관련 흐름을 빠르게 파악하고,
개인 성장과 행동으로 연결하는 것을 목표로 한다.

## 먼저 참고할 공통 문서
- `../shared/AGENTS.md`
- `../shared/config/categories.md`
- `../shared/config/scoring_rules.md`
- `../shared/docs/output_rules.md`

## X 프로젝트 역할
- 회사 시간에는 수집, 분류, 정리에 집중한다.
- 집에서는 해석, 행동 제안, 기록 자산화에 집중한다.
- X에서는 속도, 반응, 업계 분위기, 실사용자 관점을 우선적으로 본다.

## X 수집 원칙
- 광고성, 밈, 단순 반응성 글은 제외한다.
- 링크, 수치, 사건, 명확한 주장, 구체적 인사이트가 담긴 글을 우선한다.
- 계정 기반 수집과 키워드 기반 수집을 함께 사용한다.
- 중복은 제거하되, 관점이 다르면 함께 남긴다.
- 새로 발견한 좋은 계정과 키워드는 `data/candidates/`에 후보로 축적한다.

## 산출물 규칙
- 결과는 사람이 빠르게 읽을 수 있는 Markdown 형식을 우선한다.
- 각 항목에는 가능하면 작성자, 링크, 요약, 카테고리, 중요도, 최신성을 포함한다.
- 문서 마지막에는 항상 요약 섹션과 Diagnostics 섹션을 넣는다.
