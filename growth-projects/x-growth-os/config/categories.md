# categories.md

이 프로젝트는 공통 카테고리 기준으로 `../shared/config/categories.md`를 사용한다.
아래는 X 수집에서 자주 보는 세부 힌트다.

- 경제: 거시경제, 금리, 인플레이션, 환율, 에너지 가격, 정책 반응
- 투자: 실적, 가이던스, 시장 반응, 밸류에이션, 반도체, 기업 이슈
- AI: 모델 출시, API 업데이트, 에이전트, 자동화, 코딩 워크플로, 제품 적용
- 커리어: 개발 생산성, 경력 성장, 포트폴리오, 개발 문화, 업무 방식
- 기타: 위 분류에 바로 넣기 어렵지만 추후 분석 가치가 있는 항목
