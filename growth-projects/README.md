# Growth Projects

## 목적
이 폴더는 소스별로 분리된 성장 수집 프로젝트들을 모아두는 작업 공간이다.
각 프로젝트는 독립적으로 수집과 분석을 수행하고,
공통 카테고리, 스코어링, 출력 규칙은 `shared/`에서 함께 관리한다.

## 현재 프로젝트
- `shared/`: 공통 규칙, 템플릿, 명명 규칙
- `x-growth-os/`: X 기반 빠른 신호 수집과 관점 파악
- `news-growth-os/`: 뉴스 기사 기반 사실 정리와 검증
- `youtube-growth-os/`: YouTube 기반 설명형 콘텐츠 수집과 학습 가치 정리
- `integrated-growth-os/`: X, 뉴스, YouTube를 묶는 통합 클러스터링과 종합 판단

## 확장 방향
나중에 필요하면 아래 프로젝트를 같은 방식으로 추가한다.
- `market-data-os/`
- `youtube-growth-os/`
- `github-signal-os/`

## 운영 원칙
1. 소스별 수집기는 분리한다.
2. 공통 규칙은 `shared/`에서 유지한다.
