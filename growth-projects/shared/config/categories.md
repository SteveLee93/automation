# categories.md

- 경제: 거시경제, 금리, 인플레이션, 환율, 원자재, 에너지, 정책, 산업 구조 변화
- 투자: 기업 실적, 가이던스, 시장 반응, 밸류에이션, 자금 흐름, 투자 아이디어
- AI: 모델 출시, API 업데이트, 에이전트, 자동화, 추론, 제품화, 개발 워크플로
- 커리어: 개발 생산성, 경력 성장, 채용, 포트폴리오, 실무 적용, 팀 협업
- 기타: 위 카테고리에 명확히 들어가지 않지만 보존 가치가 있는 항목

## 사용 규칙
- 가능한 한 위 다섯 카테고리 안에서 분류한다.
- 애매하면 주된 가치가 가장 큰 카테고리 하나를 우선 부여한다.
- 소스별 프로젝트는 공통 이름을 그대로 사용한다.
