# source_quality_rules.md

## 최우선 출처
- 공식 기관, 공식 회사 계정, 규제기관, 원문 발표
- 1차 보도 또는 직접 취재 기반의 신뢰도 높은 기사

## 우선 출처
- 주요 뉴스 매체
- 전문 분야 해설자
- 업계 실무자 또는 제품 제작자

## 보조 출처
- 큐레이터, 요약 계정, 해설 스레드
- 개인 의견이지만 근거 링크와 맥락이 충분한 글

## 주의 출처
- 출처 없는 재가공
- 제목만 자극적이고 본문 정보가 빈약한 기사
- 광고성, 밈, 맥락 없는 반응

## 사용 규칙
- 같은 주장이라면 더 직접적인 출처를 우선한다.
- 중복 보도는 대표 출처 하나를 중심으로 남기고 나머지는 보조 근거로만 다룬다.
