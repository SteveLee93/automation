# AGENTS.md

## 프로젝트 목적
이 프로젝트는 YouTube 기반 정보 수집과 저녁 분석을 통해
경제, 투자, AI, 커리어 관련 흐름을 긴 설명형 콘텐츠 관점에서 이해하고,
학습과 행동으로 연결하는 것을 목표로 한다.

## YouTube 프로젝트 역할
- 회사 시간에는 영상 후보 수집, 품질 평가, 분류, 구조화에 집중한다.
- 집에서는 설명 깊이, 학습 가치, 실행 가능성을 중심으로 다시 해석한다.
- YouTube에서는 채널 신뢰도, 영상 형식, 설명 밀도, 학습 가치가 중요하다.

## YouTube 수집 원칙
- 제목만 보지 말고 가능하면 영상 설명, 요약, 자막 정보까지 함께 참고한다.
- 낚시성 제목, 과도한 Shorts, 반응형 클립은 우선순위를 낮춘다.
- 뉴스 브리핑형, 해설형, 시장 분석형, 인터뷰형, 실전 튜토리얼형 영상을 우선한다.
- 단순 의견보다 데이터, 사례, 원문, 공식 발표를 언급하는 영상을 우선한다.
- 새로 발견한 좋은 채널과 주제는 `data/candidates/`에 따로 축적한다.

## 산출물 규칙
- 각 항목에는 영상 제목, 채널명, 시각, 최신성, 지역, 카테고리, 형식, 신뢰도, 중요도를 포함한다.
- 수집 문서에는 Quick Notes와 Diagnostics 섹션을 반드시 넣는다.
